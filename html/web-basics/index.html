How to enjoy running

The key to enjoying running is to be able to run without any pain. There are several things that you can do to allow this to happen:

  • stretch
  • Wear the proper shoes
  • Don't over train

Some other things that can help you enjoy running are:

  1. Good music
  2. A nice place to run
  3. Nice running weather

one of the most common injuries from running are shin splints. If you do get a shin splint STOP RUNNING! If you continue running with a shin splint it will get a lot worse. The best thing you can do is REST!
